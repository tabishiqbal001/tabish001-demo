# Tabish001 Demo - Numerical Methods Implementations

This repository contains C and C++ implementations of various numerical methods for solving equations and other computational problems.

## Methods Included

- **Newton-Raphson Method**: For finding roots of nonlinear equations.
  - `newton.c`
  - `newton.cpp`
  - `newton2.cpp`

- **Regula Falsi Method**: For root finding using bracketing.
  - `falsi.cpp`
  - `falsi2.c`

- Other implementations: `16feb.c`, `23feb.cpp`, `23feb2.cpp`, `9feb.c`, `new_os.c`, `new_os1.cpp`, `new_os2_1.c`, `new_os2.cpp`, `problem1.c`

## Compilation and Running

For C files, use gcc:
```
gcc filename.c -o output -lm
./output
```

For C++ files, use g++:
```
g++ filename.cpp -o output
./output
```
